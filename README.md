------- Como Rodar a aplicação 

No terminal, clone o projeto :arrow_forward:

    - git clone https://github.com/thiagofeldner/projeto_oficina_2.0.git
    
Entre na pasta do projeto:

    - cd projeto_oficina_2.0

Instale as dependencias:

    - php artisan migrate

Execute a aplicação :

    - php artisan serve
